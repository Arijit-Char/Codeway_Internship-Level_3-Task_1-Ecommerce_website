# e-commerce-store

 ## Technology Which I Used
   ***Backend***
   
 - Nodejs
 - Mongodb
 - Express

 ***Frontend***
 

 - React Js
 - Redux

<br />

 ## Step to Start Project
 

**For Backend** 
 1. `npm i`
 2. `npm run data:import`
 3. `npm run dev`

**For Frontend**

 1. `cd e-comerce-frontend`
 2. `npm i`
 3. `npm run start`
